# 🍎 水果小丑牌 (Fruit Joker)

> 基于"小丑牌"(Balatro) 核心机制的 H5 休闲益智小游戏。在 10×10 的水果矩阵中选择 3×3 区域，策略性地搭配分数水果与倍率水果，完成关卡挑战。

---

## 快速开始

```bash
npm install
npm run dev        # 开发模式
npm run build      # 生产构建
```

---

## 核心玩法

### 基本流程

```
选择关卡 → 进入 10×10 矩阵 → 拖拽 3×3 框选择区域 → 确认选择 → 计算得分
    ↓                                                           ↓
  重复直到用完选择次数（或提前达到目标）                           ↓
    ↓                                                           ↓
  总分 ≥ 目标 → 通关结算 → 商店购物 → 下一关                      ↓
  总分 < 目标 → 失败（-1 生命，可重试）                           ↓
```

### 选择规则

- 在 **10×10 矩阵** 中使用 **3×3 选择框** 框选水果
- 选择框左上角合法范围：行 0~7，列 0~7
- 每关选择次数按大关递增：

| 大关 | 选择次数 |
|------|---------|
| 第 1~2 大关 | 3 次 |
| 第 3~4 大关 | 4 次 |
| 第 5~8 大关 | 5 次 |

- 已选中的格子**不可重复选择**
- 提前达到目标分数时，剩余次数可**转化为金币**（每次 ×2）

## 得分计算

### 计算公式

```
单次得分 = 基础分数总和 × (1 + 加法倍率) × 乘法倍率

其中：
  - 基础分数总和 = 区域内所有分数型水果基础值之和
  - 加法倍率 = 区域内所有倍率+型水果倍率之和
  - 乘法倍率 = 区域内所有倍率×型水果倍率之积
  - 特殊型水果按各自规则处理（双倍、随机等）
```

### 计算示例

区域内包含：3 个苹果 + 2 个香蕉 + 1 个车厘子 + 3 个橙子

```
基础分数总和 = 3×10 + 3×25 = 105
加法倍率     = 2×3 = 6
乘法倍率     = 1.1
最终倍率     = (1 + 6) × 1.1 = 7.7
单次得分     = 105 × 7.7 ≈ 809 分
```

### 特殊效果处理

- **毒果** ☠️：区域内包含毒果 → 该次得分 = 0
- **桃子** 🍑：区域内包含桃子 → 该次得分 ×2
- **草莓** 🍓：50% 概率触发双倍得分
- **柠檬** 🍋：降低加法倍率（负数效果）

## 关卡系统

### 关卡结构

8 大关 × 3 小关 = 共 **24 个关卡**

| 小关类型 | 说明 |
|---------|------|
| 第 1 小关 (X-1) | 纯分数挑战，无限制效果 |
| 第 2 小关 (X-2) | 1 个随机限制效果 |
| 第 3 小关 (X-3) | 2 个随机限制效果（叠加） |

### 关卡目标分数

| 大关 | 关卡 | 目标 | 新水果 | 选择次数 |
|------|------|------|--------|---------|
| 1 | 1-1 ~ 1-3 | 300 / 450 / 900 | - | 3 |
| 2 | 2-1 ~ 2-3 | 600 / 900 / 1,800 | 葡萄 | 3 |
| 3 | 3-1 ~ 3-3 | 960 / 1,440 / 2,880 | 橙子、柠檬 | 4 |
| 4 | 4-1 ~ 4-3 | 2,200 / 3,000 / 4,000 | 猕猴桃 | 4 |
| 5 | 5-1 ~ 5-3 | 5,500 / 7,500 / 10,000 | 西瓜 | 5 |
| 6 | 6-1 ~ 6-3 | 13,500 / 18,000 / 24,000 | 草莓、桃子 | 5 |
| 7 | 7-1 ~ 7-3 | 32,000 / 42,000 / 56,000 | 全部 | 5 |
| 8 | 8-1 ~ 8-3 | 74,000 / 96,000 / 400,000 | 全部 | 5 |

## 限制效果

第 2、3 小关会随机附加限制效果，增加挑战性：

| 效果 | 描述 | 出现 |
|------|------|------|
| 水果迷雾 | 随机隐藏 30% 水果显示为 ❓ | 第 2 大关起 |
| 苹果禁忌 | 矩阵中不出现苹果 | 第 2 大关起 |
| 车厘子禁忌 | 矩阵中不出现车厘子 | 第 3 大关起 |
| 香蕉禁忌 | 矩阵中不出现香蕉 | 第 3 大关起 |
| 缩小选择 | 选择框缩小为 2×2 | 第 3 大关起 |
| 时间压力 | 每次选择限时 15 秒 | 第 4 大关起 |
| 冰冻区域 | 随机 20% 格子被冻结不可选 | 第 5 大关起 |
| 水果洗牌 | 每次选择后未选区域随机重排 | 第 5 大关起 |
| 倍率反转 | 倍率效果取反（+变-，×变÷） | 第 6 大关起 |
| 毒果陷阱 | 5% 格子变为毒果 ☠️ | 第 6 大关起 |
| 全盲模式 | 全部显示为 ❓，悬停短暂显示 | 第 7 大关起 |
| 额外目标 | 除分数外需满足额外条件 | 第 7 大关起 |
| 一击必杀 | 仅 1 次选择机会 | 第 8 大关起 |

## 金币系统

### 获取方式

| 来源 | 数量 | 说明 |
|------|------|------|
| 初始金币 | 5 枚 | 新游戏开始 |
| 提前通关 | 剩余次数 × 2 | 未用完选择次数 |

| 完美通关 | 5 枚 | 单次选择即通关 |
| 利息 | floor(金币/5)，上限 10 | 每关结算时 |

### 用途

- 商店购买永久加成（强化水果数值）
- 商店购买概率提升
- 商店购买一次性道具
- 商店提前解锁高级水果
- 购买生命（5 金币 / 条）
